Welcome to Metar JSON Web API!

Please point the browser to url http://127.0.0.1:8000/metar/ping to get ping response.

Please point the browser to url http://127.0.0.1:8000/metar/info?scode=KSGS to get report data in json response. Where 'KSGS' is station code so you can use any of the station codes.

Please point the browser to url with params like http://127.0.0.1:8000/metar/info?scode=KSGS&nocache=0 to get report data in json response fetching from local cache if exists.

Please point the browser to url with params like http://127.0.0.1:8000/metar/info?scode=KSGS&nocache=1 to get report data in json response fetching live values.
